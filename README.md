## Yozech-Construction
Yosech Construction Web Platform
#Overview

The Yosech Construction Web Platform is a web-based system designed to enhance the operations, project management, and client communication of Yosech Construction, a construction company based in Dipolog City.
The platform provides an online presence where clients can explore company projects, equipment rentals, submit applications, and track project updates through a secure login system.

The system modernizes traditional workflows by centralizing project and equipment information while improving accessibility, efficiency, and customer satisfaction.
